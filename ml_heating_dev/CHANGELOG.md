# Changelog - ML Heating Control (Development)

## [0.1.0-alpha.34] - 2024-11-29

### Added
- Development release of ML Heating Control addon
- Physics-based machine learning heating control system
- Online learning capabilities with enhanced debugging
- Dashboard interface for monitoring and control
- Multi-lag learning for external heat sources
- Seasonal adaptation features
- Integration with Home Assistant climate entities
- InfluxDB integration for data storage and analysis
- Development API enabled for external access
- Debug logging enabled for troubleshooting
